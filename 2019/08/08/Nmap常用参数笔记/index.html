Nmap常用参数笔记

发表于 2019-08-08 更新于 2019-10-05 分类于 学习笔记 阅读次数： 评论数：

0x01主机发现（基本功能：探测主机存活）

nmap -sP ip段 （ ping扫描做出响应的主机）

nmap -sn ip段 （ 测试过对方主机把icmp包都丢弃掉）

nmap -sA ip段 （ 发送tcp的ack包进行探测）

0x02端口扫描（基本功能：扫描端口）

nmap -p0-65535 ip

  • （TCP）

nmap -sS ip 半开放扫描 SYN扫描 （非3次握手的tcp扫描）

nmap -sT ip 3次握手方式tcp扫描 （等同于nmap ip）

  • （UDP）

nmap -sU ip

0x03版本扫描

nmap -O ip 扫描操作系统版本

nmap -sV ip 扫描服务器版本

nmap -T 4 -A ip 综合扫描 扫描出开放端口、服务器版本、系统版本 -t 表示时间

nmap -v -Pn ip 无ping扫描 有些防火墙禁止ping 可以使用此选项 -v 显示详细扫描信息

nmap --script <脚本名> 192.168.198.131 用脚本扫描是否存在漏洞

0x04输出参数

参数
-oN 标准保存
-oX XML保存
-oG Grep保存
-oA 保存到所有格式
-append-output 补充保存文件
